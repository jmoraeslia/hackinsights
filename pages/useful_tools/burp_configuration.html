Burp Suite (community edition)

This framework is a useful tool that pentesters or security pros (and maybe some great developers) use to test certain applications. We don't need to spend so much time explaining and throwing a lot of theoretical information here. Instead, what I wanna do is step into the practical and make sense of the theory through this active learning process.

Before, read this

I know that is hard to concentrate and learn new things, It's overwhelming and you need to overcome this. I promisse that it'll worth it.

How?

What I'm going to say here is valid for EVERYTHING you want to learn and understand: FOCUS ON PRINCIPLES and overlook the rest - you'll learn most of it automatically while doing the work. This is practice.
Remember: initially, ignore what is not covered; just set up your environment and hack! (In a secure lab, obviously..)

Install

If you are using Kali Linux (and i recommend that), you already have the Burp pre-installed. If not, go to this link and download the community edition. After installing, we need to configure our browser to accept the interceptions of our requests in order to manipulate them (changing data, mostly).

Configure

There are several ways to do this. but i will presume here you're using Firefox and i will set up the proxy from the FoxyProxy extension. Download it here.

After installing, click in the puzzle icon on the right superior corner of the browser:

  • title: burp
  • hostname: 127.0.0.1
  • port: 8080

I would recommend you pin this extension to make it easier to connect to the proxy.
